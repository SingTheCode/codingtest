### 빈 배열 확인 [practice1.js](practice1.js)

빈 배열 자체는 true를 반환한다. 따라서 배열의 길이로 빈 배열 여부를 확인한다.

```js
const arr = [];

console.log(!!arr); // true
console.log(Array.isArray(arr) && arr.length) // false
```
